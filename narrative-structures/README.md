## 結構類型總覽

### 依分支程度

| 類型 | 特徵 |
|------|------|
| 線性 | 無選擇或選擇不影響結局 |
| 輕度分支 | 少量關鍵選擇，2-3 結局 |
| 重度分支 | 大量選擇，多條劇情線 |
| 路線制 | 角色路線，共通線 + 個別線 |

### 依時間結構

- 單一時間線
- 多視角切換
- 時間迴圈
- 非線性時間

### 依長度

| 分類 | 時數 |
|------|------|
| 短篇 | < 2 小時 |
| 中篇 | 2-10 小時 |
| 長篇 | 10-30 小時 |
| 超長篇 | 30+ 小時 |

### 特殊形式

- 章節式發行
- 選集形式
- 迴圈 / Roguelike 敘事
