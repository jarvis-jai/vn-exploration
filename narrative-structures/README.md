# 敘事結構（Narrative Structures）

視覺小說的敘事結構決定玩家如何經歷故事。本目錄整理各種敘事設計模式。

---

## 這裡收錄什麼

- 分支結構類型
- 線性 vs 非線性設計
- 選擇機制設計
- 章節與節奏安排
- 多結局設計模式

---

## 結構類型（待填充）

### 依分支程度

- **線性敘事**（Linear）
  - 無選擇或選擇不影響結局
  - 純閱讀體驗

- **輕度分支**（Light Branching）
  - 少量關鍵選擇
  - 2-3 個主要結局

- **重度分支**（Heavy Branching）
  - 大量選擇點
  - 多條劇情線
  - 需要多周目才能看完

- **平行路線**（Route System）
  - 角色路線制（常見於戀愛 VN）
  - 共通線 + 個別線

### 依時間結構

- **單一時間線**
- **多視角切換**
- **時間迴圈**（Loop）
- **倒敘／非線性時間**

### 依長度

- **短篇**（< 2 小時）
- **中篇**（2-10 小時）
- **長篇**（10-30 小時）
- **超長篇**（30+ 小時）

### 特殊形式

- **章節式發行**（Episodic）
- **選集形式**（Anthology）
- **無限迴圈 / Roguelike 敘事**
