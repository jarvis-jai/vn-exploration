# 粉絲創作生態：從同人誌到 Fan Game

視覺小說的粉絲創作（二次創作、fan work）構成了一個龐大且複雜的生態系統。這個生態系統不僅是「消費者的自娛」，更是 VN 產業的重要組成部分——許多商業作品的創作者本身就是從同人起步，而粉絲社群的活躍程度往往決定了一部作品的文化影響力與壽命。

---

## 粉絲創作的類型光譜

### 文本類創作

**同人小說（二次創作小說）：**
- SS（Short Story）：短篇同人故事
- 長篇連載
- IF 故事：「如果走了另一條路」
- 跨作品 crossover
- 性轉、現代 AU 等變體設定

**主要發表平台：**
- pixiv（日本，含小說功能）
- ハーメルン、小説家になろう（日本，原創與二創混合）
- Archive of Our Own（AO3，西方主導）
- 中國的 LOFTER、晉江等

**文本同人的特殊性：**
- VN 本身是文本媒介，同人小說與原作形式接近
- 可以「補完」原作沒有的路線
- 但也因此更容易被拿來與原作比較
- 「同人寫得比官方好」的評價（褒貶兼具）

### 視覺類創作

**同人誌（漫畫、畫集）：**
- Comic Market（Comiket）是最大的同人即賣會
- VN 角色的二創插畫
- 漫畫形式的同人故事
- R-18 創作（占相當比例）

**插畫與 fanart：**
- pixiv 是主要集散地
- Twitter/X 上的日常創作
- 西方則以 DeviantArt、Tumblr、Twitter 為主
- 近年 AI 生成圖像引發的爭議

**Cosplay：**
- VN 角色的 cosplay 文化
- 與動畫化後的普及程度相關
- 同人即賣會的重要組成
- 「還原度」vs「詮釋」的社群標準

### 音樂類創作

**同人音樂：**
- 原作 BGM 的 arrange 版本
- Vocal 化（將 BGM 加上歌詞演唱）
- 角色印象曲（image song）
- 同人 vocal 社團的活動

**主要平台與通路：**
- 同人即賣會實體販售
- Booth、Bandcamp 數位販售
- YouTube、niconico 試聽

**VN 音樂同人的特殊性：**
- VN 音樂本身就有較高辨識度
- 「神曲」效應：鳥の詩、届かない恋 等
- 音樂同人往往跨作品

### 遊戲類創作（Fan Game）

**Fan Game 的類型：**

1. **補完型：**
   - 製作原作沒有的路線
   - 「如果 XX 有自己的路線」
   - After Story / 後日談

2. **平行型：**
   - 使用原作世界觀但不同主角
   - 原創角色（OC）在原作世界
   - Crossover 作品

3. **遊戲化型：**
   - 將 VN 改編成其他類型
   - 例如：格鬥遊戲、RPG、音遊
   - Melty Blood（TYPE-MOON 授權但起源於同人）

4. **致敬/戲仿型：**
   - 搞笑、惡搞性質
   - 對原作的元敘事評論
   - Meme 文化的延伸

**Fan Game 的技術：**
- Ren'Py 是最常用的引擎
- 部分使用原作引擎（如 KiriKiri 的腳本修改）
- 美術資源的處理（借用、自製、委託）

## 同人即賣會與流通

### Comic Market（Comiket）

**規模與影響：**
- 一年兩次（夏、冬）
- 參加者數十萬人
- 攤位數萬
- 日本同人文化的核心

**VN 相關攤位的分布：**
- 遊戲區（ゲーム系）
- 依作品分類（Fate 區、Key 區等）
- 大型 IP 有專屬區域
- 小眾作品則在一般區域

**攤位的類型：**
- 繪師個人社團
- 小說社團
- 音樂社團
- 遊戲（Fan Game）社團
- 複數創作者的合同誌

### 其他即賣會

**例大祭（博麗神社例大祭）：**
- 東方 Project 專門
- 雖然東方嚴格說不是 VN，但影響了 VN 同人文化
- 「二次創作自由」的模式被引用

**其他作品專門即賣會：**
- Fate 系有專門的即賣會
- Key 相關的即賣會
- 依 IP 熱度而生滅

**西方的 con 文化：**
- Anime Expo、Anime NYC 等
- 同人攤位（Artist Alley）
- 規模較小，形式不同
- 較少 VN 專門的同人

### 數位流通

**Booth（pixiv 運營）：**
- 實體與數位商品
- 同人誌、音樂、遊戲
- 手續費相對低
- 國際購買可行（有門檻）

**DLsite 同人區：**
- 以數位內容為主
- 成人內容比例高
- 同人遊戲的重要平台

**itch.io：**
- 西方同人遊戲常用
- 「Pay what you want」模式
- 法律灰色地帶的作品較多

## 法律與默許：二創的灰色地帶

### 日本的「默認許可」文化

**版權的嚴格與寬鬆並存：**
- 法律上，二創未經授權是侵權
- 但實務上，多數權利方「睜一隻眼閉一隻眼」
- 形成了一種「默認許可」的文化

**默許的條件（不成文規則）：**
- 不營利或僅小規模營利
- 標註是「同人」「二次創作」
- 不冒充官方
- 不損害作品形象（標準模糊）
- 尊重創作者意願

**官方的態度光譜：**

| 態度 | 描述 | 例子 |
|------|------|------|
| 積極鼓勵 | 明確聲明允許二創 | 東方 Project |
| 沉默默許 | 不聲明但不取締 | 多數 VN |
| 有限許可 | 提供官方指引 | 部分大型 IP |
| 選擇性執法 | 對某些類型取締 | 依情況 |
| 嚴格禁止 | 主動取締 | 少數，通常是非日本權利方 |

**「官方二創指南」的案例：**
- TYPE-MOON 有針對 Fate 系的二創指引
- Cygames（不是 VN 但模式可參考）的指引
- 指引通常規範：商業使用、R-18、冒充官方等

### 敏感地帶

**成人內容（R-18）：**
- 角色的成人化二創
- 原作全年齡 → 二創 R-18
- 原作已是成人向 → 程度加重
- 部分創作者對此反感
- 但整體是被默許的

**商業化的界線：**
- Comiket 的「小規模」販售通常 OK
- 但大量生產、商業渠道販售可能觸紅線
- 「同人是愛好，不是生意」的說法
- 實際上有人以同人為主要收入

**跨公司 crossover：**
- 多個作品的角色在同一創作中
- 版權歸屬複雜
- 通常也是默許
- 但公開販售時更謹慎

**AI 生成內容的新爭議：**
- AI 繪圖生成的二創
- 「這算創作嗎？」
- Comiket 對此的立場演變
- 繪師社群的強烈反彈

### 西方的情況

**「Fair Use」的不確定性：**
- 美國版權法的 Fair Use 條款
- 但二創（尤其是營利性）不一定符合
- 法律風險較日本大

**DMCA Takedown：**
- 權利方可以發 takedown notice
- 對西方同人社群影響大
- 日本公司對西方作品較少執法

**不同平台的政策：**
- AO3 以保護創作自由聞名
- DeviantArt、Tumblr 有各自政策
- itch.io 對 fan game 態度寬鬆

## 從同人到商業：畢業的路徑

### 「同人→商業」的經典案例

**TYPE-MOON：**
- 《月姬》是同人遊戲起步
- Comiket 販售累積名氣
- 成立公司後的《Fate/stay night》商業發售
- 成為 IP 帝國
- 最經典的「同人畢業」案例

**07th Expansion（竜騎士07）：**
- 《寒蟬鳴泣之時》同人發行
- 持續以同人形式運營
- 但影響力達到商業級
- 選擇不完全「商業化」的另一種路徑

**Key 的部分成員：**
- 有些成員有同人背景
- 但 Key 本身是商業公司起步
- 與 TYPE-MOON 路徑不同

**西方案例：**
- 較少「同人→商業 VN」的直接案例
- 但 indie 開發者可能有 fan game 經歷
- Toby Fox（Undertale）的 Homestuck 二創背景

### 同人作為「試金石」

**驗證市場的功能：**
- 同人的受歡迎程度預示商業潛力
- 先以同人測試風格、故事
- 獲得反饋後再商業化
- 降低風險的策略

**累積技術與經驗：**
- 同人製作是練習
- 學習工具、流程
- 建立團隊
- 瞭解玩家反應

**建立粉絲基礎：**
- 同人創作者有自己的追隨者
- 商業化時自帶初始受眾
- 口碑傳播的基礎

### 「永遠的同人」路線

**不商業化的選擇：**
- 維持同人形式的自由
- 不需面對商業壓力
- 保持與粉絲的直接關係
- 竜騎士07 的某種程度選擇

**同人的優勢：**
- 創作自由度高
- 不需迎合市場
- 失敗成本低
- 可以實驗

**同人的限制：**
- 規模受限
- 收入不穩定
- 難以全職投入（對多數人而言）
- 法律風險（雖然低）

## 粉絲創作對原作的影響

### 正面影響

**延長作品壽命：**
- 原作完結後，粉絲創作維持熱度
- 新玩家通過二創發現原作
- 「經典」地位的維持

**擴大受眾：**
- 二創作為入口
- 「看了同人才去玩原作」
- 跨圈傳播

**社群凝聚：**
- 二創是社群活動的核心
- 創作者與消費者的互動
- 維持「活著的」社群

**反饋與對話：**
- 二創反映玩家的解讀
- 官方可以觀察玩家偏好
- 某些二創梗被官方吸收

### 負面/複雜影響

**形象脫離原作：**
- 二創中的角色與原作不同
- 新玩家可能有錯誤預期
- 「meme 化」導致角色被簡化

**劇透問題：**
- 二創可能暴露關鍵劇情
- 特別是名場面的二創
- 「還沒玩就被劇透」

**品質參差：**
- 劣質二創可能損害印象
- 「官方最大的敵人是部分粉絲」
- 但這是所有 IP 的共同問題

**R-18 二創的爭議：**
- 部分玩家對此反感
- 「為什麼要這樣對待她」
- 創作者對角色被 R-18 化的態度
- 沒有共識，是持續的爭議

### 官方與同人的互動

**官方吸收同人梗：**
- 某些二創設定被官方採用
- Fandisc 或續作中的彩蛋
- 「官方認證」的狂喜

**官方邀請同人作者：**
- 商業作品邀請知名同人畫師
- 官方畫集收錄同人作品
- 模糊「官方/同人」界線

**同人作者成為官方：**
- 被挖角進入公司
- 或者接受官方委託
- 身分轉換的複雜性

## 特定 IP 的同人生態

### TYPE-MOON 系（Fate/月姬等）

**規模：**
- 可能是最大的 VN 同人生態
- Comiket 有專屬大區域
- 同人遊戲、漫畫、小說、音樂俱全

**特徵：**
- 設定厨文化——深挖設定的二創
- 跨作品（Fate/月姬/魔法使之夜）
- FGO 帶來的新一波創作者
- 聖杯戰爭的框架適合二創

**官方態度：**
- 有明確的二創指引
- 總體寬容
- 但對商業侵權會執法

### Key 系（CLANNAD/AIR/Little Busters等）

**特徵：**
- 「感動系」二創——延續原作情感
- After Story 型的補完
- 較少「魔改」
- 角色的「純潔」形象被較多維護

**音樂二創：**
- Key 音樂的 arrange 非常豐富
- 同人 vocal 社團活躍
- 「鳥の詩」的無數版本

### 東方 Project（參考）

**雖非 VN 但影響巨大：**
- ZUN 明確開放二創
- 成為日本同人文化的核心
- 影響了整個二創生態的「規範」
- VN 社群參考其模式

**東方模式的特點：**
- 原作提供框架，二創自由發揮
- 官方角色設定簡約，留給二創空間
- 商業化有明確限制
- 音樂、遊戲、漫畫全方位

## 二創社群的內部文化

### 「圈子」與「圈規」

**圈子的劃分：**
- 依作品分
- 依 CP（配對）分
- 依創作類型分
- 圈子之間有時有張力

**CP 文化與爭議：**
- 角色配對是二創的核心主題之一
- 「官配」vs「非官配」
- 「逆」（攻受互換）的爭議
- 「CP 黨」之間的衝突

**不成文規則：**
- 「不要在官方帳號下發 R-18」
- 「標註 CP 和警告」
- 「不要攻擊其他 CP」
- 這些規則因圈子而異

### 創作者之間的關係

**社團的組成：**
- 通常是朋友或網友
- 分工（繪師、寫手、排版等）
- 小規模、彈性
- 可能因作品熱度變化而重組

**委託與合作：**
- 創作者之間互相委託
- 合同誌（多人合作的同人誌）
- 跨社團的企劃

**競爭與認可：**
- 在社群內的「地位」
- follower 數、銷量、評價
- 被官方認可是最高榮譽
- 也有嫉妒與紛爭

### 創作者的動機

**為愛發電：**
- 「因為喜歡所以創作」
- 不期待回報
- 創作本身是目的

**社群參與：**
- 成為社群的一部分
- 與同好交流
- 共同的熱情

**技術練習：**
- 提升創作能力
- 為未來的商業創作準備
- 建立作品集

**經濟因素：**
- 部分創作者以此為副業
- 極少數以此為主業
- 但多數不以營利為主要目的
