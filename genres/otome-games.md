# 乙女遊戲（Otome Games）

女性向戀愛 VN 的專論。乙女遊戲作為 VN 生態的重要支柱，擁有獨立的產業結構、設計語法、社群文化，與男性向 VN 形成有趣的對照與互補。

---

## 定義與邊界

### 什麼是乙女遊戲

**基本定義：**
- 女性玩家為主要目標對象
- 女性主角與男性角色之間的戀愛為核心
- 玩家透過選擇推進與不同男性角色的關係

**日文詞源：**
- 「乙女」（おとめ）= 少女、處女、年輕女性
- 「乙女ゲーム」= 少女遊戲
- 常簡稱「乙ゲー」（otoge）

**與其他類型的區分：**

| 類型 | 主角性別 | 攻略對象 | 主要受眾 |
|------|---------|---------|---------|
| 乙女遊戲 | 女 | 男 | 女 |
| 美少女遊戲 | 男 | 女 | 男 |
| BL 遊戲 | 男 | 男 | 女（為主） |
| GL 遊戲 | 女 | 女 | 不定 |

### 邊界模糊區域

**乙女 × BL 混合：**
- 部分作品同時包含男性可攻略角色與男男路線
- 「夢女子」（想與角色戀愛）vs「腐女子」（想看男男戀）的受眾區分
- 有時同一 IP 會出不同版本

**乙女 × 遊戲性融合：**
- RPG 元素（如 Hakuoki 的戰鬥系統）
- 經營模擬（如 Tokimeki Memorial Girl's Side）
- 音樂遊戲（如 うたの☆プリンスさまっ♪）

**女性向 ≠ 乙女：**
- 並非所有女性向遊戲都是戀愛遊戲
- 女性向泛指目標受眾為女性
- 乙女遊戲是女性向的子集

## 歷史脈絡

### 起源（1990 年代）

**公認的開創性作品：**
- **アンジェリーク（Angelique）**（1994，光榮）
  - 普遍被視為乙女遊戲的始祖
  - SFC 平台
  - 設定：女王候選人與九位守護聖
  - 確立了「多位可攻略男性 + 女主角選擇」的基本形式

**早期發展環境：**
- 當時遊戲市場以男性為主
- 女性玩家被視為邊緣市場
- 光榮的「Ruby Party」團隊專門開發女性向遊戲

### 成長期（2000 年代）

**平台擴展：**
- PS2 時代作品數量爆發
- PSP 成為乙女遊戲重要平台
- DS 也有不少作品

**代表作品湧現：**
- **薄桜鬼（Hakuoki）**（2008）：新撰組題材，跨媒體展開
- **うたの☆プリンスさまっ♪（UtaPri）**（2010）：偶像題材，音樂元素
- **AMNESIA**（2011）：失憶症題材，獨特美術風格

**品牌確立：**
- Otomate（Idea Factory 旗下）成為最大品牌
- 音泉、Rejet 等專門廠商成立
- 形成獨立的產業生態

### 手遊時代（2010 年代至今）

**遊戲模式轉變：**
- 從買斷制轉向 F2P + gacha
- 「戀愛遊戲」手遊大量出現
- 角色卡牌收集成為主要機制

**代表性手遊：**
- **イケメンシリーズ**（Cybird）：多款同系列遊戲
- **夢100（夢王國與沉睡的100位王子殿下）**
- **あんさんぶるスターズ！！（偶像夢幻祭）**
- **ツイステ（Twisted Wonderland）**

**手遊化的影響：**
- 觸及更廣受眾
- 劇本被碎片化
- 消費模式改變（持續付費 vs 一次買斷）
- 社交元素增加

## 設計語法

### 女主角設計

**常見類型：**

**「無色透明」型：**
- 個性模糊，方便代入
- 外觀標準化（有時可自訂）
- 對話選項代替固定性格

**「性格明確」型：**
- 有自己的聲音與想法
- 玩家「觀看」而非「代入」
- 近年越來越受歡迎

**設計張力：**
- 代入派 vs 觀賞派的社群分歧
- 太無個性會被批「紙片人」
- 太有個性可能讓部分玩家不適

**視覺處理：**
- 有些作品女主角沒有立繪
- 有些只顯示背影或局部
- 有些有完整設計但 CG 時隱藏面部
- 近年趨勢是提供完整視覺設計

### 男性可攻略角色設計

**角色屬性公式：**

乙女遊戲發展出一套高度類型化的角色設計語法，類似於角色扮演遊戲的「職業」系統。

**常見屬性類型：**

| 類型 | 日文 | 特徵 |
|------|------|------|
| 王子型 | 王子様系 | 高貴、溫柔、完美 |
| 俺樣型 | 俺様系 | 霸道、自信、強勢 |
| 病嬌型 | ヤンデレ | 過度執著、佔有慾強 |
| 傲嬌型 | ツンデレ | 表面冷淡、內心柔軟 |
| 弟系 | 弟系 | 年下、撒嬌、可愛 |
| 眼鏡型 | 眼鏡系 | 知性、冷靜、可能腹黑 |
| 元氣型 | 元気系 | 活潑、陽光、單純 |
| 不良型 | 不良系 | 叛逆、外冷內熱 |

**角色配置原則：**
- 一款遊戲通常涵蓋多種屬性
- 確保不同玩家偏好都能找到「本命」
- 角色之間的對比與互動也很重要

**外觀設計特徵：**
- 髮色多樣化（區分辨識）
- 身高差異明顯
- 服裝反映性格
- 眼睛設計特別精細

### 路線結構

**典型結構：**

1. **共通線**
   - 女主角與所有角色相遇
   - 建立基本設定與關係
   - 透過選擇累積好感度

2. **個別路線**
   - 專注於單一角色
   - 深入探索該角色的背景與內心
   - 戀愛關係發展

3. **結局分歧**
   - Good End / Normal End / Bad End
   - 有時有「真結局」需完成所有路線

**路線解鎖順序：**
- 有些遊戲自由選擇
- 有些推薦順序
- 有些強制順序（留「真相」到最後）

### 敘事慣例

**常見劇情模板：**

**學園物：**
- 轉學生女主角
- 校園社團或事件
- 同學或學長作為攻略對象

**異世界 / 幻想：**
- 穿越到異世界
- 成為重要人物（女王、巫女等）
- 守護者或騎士作為攻略對象

**歷史 / 和風：**
- 幕末、戰國等時代背景
- 歷史人物性轉或戀愛化
- 薄桜鬼的成功確立此類型

**偶像 / 娛樂圈：**
- 女主角是經紀人、作曲家等
- 攻略對象是偶像或藝人
- UtaPri 類型的延續

**現代職業：**
- 職場戀愛
- 攻略對象是同事或上司
- 較寫實的設定

**「糖度」的概念：**
- 甜度、浪漫程度
- 「高糖」= 大量甜蜜場景
- 「低糖」= 劇情為主、戀愛為輔
- 不同玩家偏好不同糖度

## 視覺與聽覺設計

### 美術風格

**與男性向的差異：**
- 男性角色設計更精緻
- 少女漫畫式的美學
- 眼睛比例、睫毛、光效
- 背景偏華麗夢幻

**CG 的重要性：**
- 戀愛高潮場景的 CG 至關重要
- 收集 CG 是重要動力
- CG 品質直接影響評價

**代表性原畫師：**
- カズキヨネ（薄桜鬼、Code:Realize）
- 倉花千夏（AMNESIA、Collar×Malice）
- 悌太（DIABOLIK LOVERS）

### 聲優（CV）文化

**聲優的核心地位：**
- 乙女遊戲的聲優幾乎全配音
- 聲優人氣直接影響銷量
- 聲優 event 是重要行銷手段

**「CV 買い」現象：**
- 玩家因為某聲優而購買遊戲
- 與內容品質無關的消費動機
- 部分遊戲的主要賣點就是聲優陣容

**常見乙女遊戲聲優（以活躍度計）：**
- 石川界人、下野紘、森久保祥太郎
- 鳥海浩輔、岡本信彦、梶裕貴
- 諏訪部順一、木村良平、宮野真守
- （此列表非排名，僅示例）

**聲優與角色的連結：**
- 特定聲優 × 特定屬性的配對期待
- 聲優的演技風格影響角色詮釋
- 系列作品會保持聲優一致性

### 音樂設計

**主題歌的重要性：**
- OP / ED 品質被高度關注
- 角色歌（Character Song）是常見商品
- 偶像系尤其強調音樂

**配樂特徵：**
- 鋼琴、弦樂為主的配器
- 浪漫、夢幻的氛圍
- 緊張場景的處理

## 商業模式

### 傳統買斷制

**平台分佈：**
- 主機：PS Vita（已停產但仍有庫存新作）、Nintendo Switch
- PC：Steam 近年增加
- 移動端：買斷制 App 較少

**價格帶：**
- 日本：主機版約 6,000-8,000 日圓
- 限定版：10,000 日圓以上（含特典）
- 折扣後：3,000-5,000 日圓

**特典商法：**
- 店舖特典（不同店舖不同特典）
- 限定版內容物（Drama CD、畫冊等）
- 收集特典是購買動機之一

### 手遊 F2P 模式

**主流商業模式：**
- 免費遊玩，gacha 抽角色卡牌
- 體力系統限制遊玩量
- 活動驅動的持續運營

**課金設計：**
- 抽卡（主要收入來源）
- 體力回復
- 特別劇情解鎖
- 服裝 / 裝飾道具

**與傳統 VN 的差異：**
- 劇情碎片化
- 持續追加內容
- 社交 / 競爭元素
- 「養成」感更強

### 跨媒體展開

**常見展開形式：**
- 動畫化
- 漫畫化
- 舞台劇
- 周邊商品
- CD（Drama CD、角色歌）
- 現場活動

**IP 價值建構：**
- 遊戲本體可能只是入口
- 周邊收入可能超過遊戲本身
- 聲優活動是重要收入來源

## 社群生態

### 日本社群

**主要據點：**
- Twitter/X：角色圖、感想、二創
- Pixiv：插圖、小說
- 各種 BBS / 討論板

**社群文化特徵：**
- 「推し」（oshi）文化：有明確的本命角色
- 大量二次創作
- 角色生日等紀念日的慶祝
- 對「解釋」女主角的討論

**「夢女子」vs「腐女子」：**
- 夢女子：想與角色戀愛（乙女遊戲主受眾）
- 腐女子：想看男性角色之間的戀愛
- 有重疊，但自我認同不同
- 社群有時會有摩擦

### 英語圈社群

**平台：**
- Reddit（r/otomegames）
- Twitter/X
- Discord

**文化差異：**
- 更重視女主角的「強勢」程度
- 對某些日本慣例的批評（如過度順從的女主角）
- localization 品質的討論
- indie otome 的關注

### 中文圈社群

**平台：**
- 微博
- Bilibili
- 各種論壇

**特徵：**
- 漢化討論
- 日本乙女 vs 國產乙女的比較
- 對「戀與製作人」等國產作品的討論

## 國際化與在地化

### 翻譯狀況

**官方英文版：**
- Aksys Games 是主要英文發行商
- Idea Factory International
- 近年 Steam 發行增加

**翻譯挑戰：**
- 敬語等級的處理
- 稱呼方式（さん、くん、呼び捨て）
- 日本特有文化元素
- 女主角語氣的定調

**民間漢化：**
- 大量民間漢化存在
- 品質參差不齊
- 有時比官方更受歡迎

### 西方 indie otome

**特徵：**
- Ren'Py 引擎為主
- 更多元的女主角與角色設定
- LGBTQ+ 選項更常見
- itch.io 是主要發行平台

**代表作品：**
- **Cinderella Phenomenon**：免費、高品質
- **Our Life: Beginnings & Always**：高度客製化
- **Hustle Cat**：咖啡店設定、輕鬆風格

**文化差異：**
- 對「俺樣」等類型的接受度較低
- 更在意女主角的 agency
- 對 problematic 元素更敏感

## 爭議與批評

### 女主角的塑造

**常見批評：**
- 過度被動
- 總是需要被拯救
- 沒有自己的目標或夢想
- 「沒有個性」

**辯護觀點：**
- 代入感設計的必然
- 後期作品已改善
- 不同玩家想要不同類型
- 不應以男性向標準評判

### 戀愛動態的問題

**常見批評：**
- 「俺樣」角色的霸道行為
- 病嬌角色的危險訊號
- 權力不對等的關係（上司-下屬等）
- 「禁斷」關係的浪漫化

**社群反應：**
- 部分玩家喜歡這些元素（虛構與現實的區分）
- 部分玩家希望更健康的關係描寫
- 世代與文化的差異

### 多樣性問題

**角色多樣性：**
- 攻略對象外貌高度標準化
- 深膚色角色稀少
- 體型多樣性缺乏

**性向多樣性：**
- 傳統乙女遊戲假設異性戀女玩家
- GL 選項在日本商業作品中罕見
- 西方 indie 更多元

## 與男性向 VN 的對照

### 結構差異

| 面向 | 乙女遊戲 | 美少女遊戲 |
|------|---------|-----------|
| 主角代入 | 中等程度 | 高度代入 |
| 攻略對象深度 | 每角色路線完整 | 差異較大 |
| 共通線長度 | 通常較長 | 變異大 |
| 聲優重要性 | 極高 | 高但次於乙女 |
| H 內容 | 少數作品有 | 常見 |
| 跨媒體 | 常見 | 變異大 |

### 市場差異

| 面向 | 乙女市場 | 美少女市場 |
|------|---------|-----------|
| 市場規模 | 較小 | 較大 |
| 手遊比例 | 極高 | 增加中 |
| 實體活動 | 聲優活動為主 | 較少 |
| 周邊商品 | 重要收入來源 | 次要 |
| 國際化程度 | 增加中 | 歷史較久 |

### 社群文化差異

| 面向 | 乙女社群 | 美少女社群 |
|------|---------|-----------|
| 二創量 | 極大 | 大 |
| 討論焦點 | 角色、聲優 | 劇本、系統 |
| 排他性 | 較低 | 較低 |
| 「推し」文化 | 核心 | 存在但不同形式 |
