## 鄰近類型概覽

### 互動小說（Interactive Fiction）

→ 詳見 [interactive-fiction.md](./interactive-fiction.md)

VN 的重要先驅與並行類型。從 1970 年代的文字冒險演化至今，包含 parser-based 與 choice-based 兩大支脈。Choice-based IF（如 Twine 遊戲）與 VN 的設計極為接近，差異主要在視覺呈現。兩個社群有不同的傳統但逐漸融合。

### 隱藏物件遊戲（Hidden Object）

- **與 VN 的關係**
  - 常有故事驅動元素
  - 敘事 + 解謎結合
  - 女性向市場重疊

- **代表系列**
  - **Mystery Case Files**（Eipix / Big Fish）— HOG 類型開創者，長篇連續劇情
  - **Enigmatis** 三部曲（Artifex Mundi）— 黑暗懸疑，故事完整度高
  - **Grim Tales** 系列（Elephant Games）— 黑暗童話改編
  - **Hidden City**（G5）— 手機 HOG 代表
  - **Artifex Mundi 作品群** — 以敘事品質著稱的 HOG 發行商

### 敘事解謎（Narrative Puzzle）

- **特徵**
  - 解謎推動劇情
  - 互動性較高

- **代表作品**
  - **Professor Layton 系列**（Level-5）— 解謎 + 敘事的經典結合
  - **Zero Escape / 極限脫出系列**（打越鋼太郎）— 密室脫出 + VN，敘事詭計
  - **Danganronpa / 彈丸論破**（Spike Chunsoft）— 推理 + 學級裁判
  - **The Room 系列**（Fireproof）— 3D 機關盒解謎
  - **Return of the Obra Dinn**（Lucas Pope）— 推理重建敘事
  - **Her Story** / **Telling Lies**（Sam Barlow）— FMV 敘事解謎
  - **The Witness**（Jonathan Blow）— 環境敘事 + 解謎

### 冒險遊戲（Adventure Game）

- **傳統點擊冒險**
  - 探索、物品收集、對話
  - 與 VN 有共同祖先

- **現代敘事冒險**
  - Walking Simulator
  - 互動電影
  - Telltale 系列作品

### 養成遊戲（Raising Sim）

→ 詳見 [raising-simulation.md](./raising-simulation.md)

與 VN 深度交織的鄰近類型。從《美少女夢工場》（1991）確立的女兒養成範式，到《心跳回憶》（1994）的戀愛養成模式，養成遊戲與 VN 在日本美少女遊戲市場有大量重疊。兩者的混合形成一個光譜：從重養成端（數值管理為核心）到輕養成端（敘事為主、數值隱性），再到純 VN（完全無養成）。現代趨勢是養成元素的輕量化與隱性化，但課金模式也帶來新的設計扭曲。

### RPG 混合

- **JRPG + VN 元素**
  - 大量對話場景
  - 角色關係系統
  - Persona 系列的對話部分

- **Dating Sim + RPG**

### 其他混合形式

- **VN + 模擬經營**
- **VN + 節奏遊戲**
- **VN + 策略**
- **VN + 卡牌**

## 設計借鑑

### 從其他類型借鑑到 VN

- IF 的敘事結構分類（Time Cave、Branch and Bottleneck 等）
- 冒險遊戲的環境探索設計
- 養成遊戲的數值可視化

### 從 VN 借鑑到其他類型

- 角色刻畫技巧影響現代 IF
- 對話框呈現方式被廣泛採用
- 路線/結局系統的影響
