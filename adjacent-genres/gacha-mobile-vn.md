# 手機遊戲與抽卡遊戲：VN 的商業變體

當代最成功的「VN 式敘事」載體，很可能不是傳統的 PC/主機視覺小說，而是手機上的抽卡遊戲（gacha game）。《Fate/Grand Order》《碧藍航線》《原神》《明日方舟》——這些遊戲繼承了 VN 的敘事方法論，但將其嵌入完全不同的商業模式與消費結構中。

本文探討這個混合領域：VN 的敘事基因如何在手遊生態中變異、存活、繁殖。

---

## 定義與邊界

### 什麼是「VN 式手遊」

**特徵光譜：**
- 敘事佔比極高（主線劇情、角色故事、活動劇情）
- 對話框呈現（立繪 + 文字框 + 語音）
- 角色作為核心賣點
- 收集與擬社會關係並存

**與傳統 VN 的差異：**
- 商業模式（買斷 vs F2P + 抽卡）
- 敘事完整性（一次性 vs 長期營運、持續更新）
- 互動結構（閱讀 vs 遊戲系統 + 閱讀）
- 消費單位（作品 vs 角色）

### 類型光譜

```

     │                                                       │
     │  傳統 VN                                               │
     │       │                                               │

     │       │       │       │       │       │               │
     │       │       │       │       │       │  輕敘事手遊    │
```

## 核心案例分析

### Fate/Grand Order（2015-）

**VN 血統：**
- 直接延續 TYPE-MOON / 奈須きのこ的創作
- 敘事風格與《Fate/stay night》一脈相承
- 主線劇情實際上就是一部超長篇 VN
- 第一部（七個特異點 + 終章）約 100 萬字以上

**敘事結構的適應：**
- 主線章節制（每章約 2-4 小時閱讀量）
- 免費節點 vs 故事節點
- 「打完這關才能看下一段劇情」
- 活動劇情作為補充敘事

**角色消費的變異：**
- 傳統 VN：支付一次，擁有所有角色路線
- FGO：角色需要「抽到」，概率極低
- 付費購買的是「相遇的機會」
- 沒抽到 = 永遠無法擁有這個角色

**擬社會關係的商業化：**
- 角色好感度系統（羈絆等級）
- 專屬禮裝（角色專屬裝備）
- 幕間物語（角色專屬劇情，需要擁有該角色）
- 情人節活動（角色給你巧克力）
- 「與角色的關係」成為持續消費的動力

**敘事品質的爭議：**
- 第一部序章：公認低迷
- 第六章（Camelot）：口碑轉折
- 第一部終章（Solomon）：情感高潮
- 第二部品質波動大
- 「後期好看但前期勸退」問題

**數據參考（截至 2024）：**
- 全球累計營收超過 70 億美元
- 日本市場長期霸榜
- 證明「重敘事手遊」可行

### 明日方舟（Arknights，2019-）

**VN 元素的運用：**
- 主線劇情文字量龐大
- 對話框 + 立繪的經典呈現
- 角色檔案作為補充敘事
- 世界觀極度詳細

**敘事風格特徵：**
- 比 FGO 更「硬」的科幻設定
- 政治、戰爭、種族議題
- 不迴避黑暗與灰色地帶
- 文本密度高，被戲稱「方舟文學」

**整合策略遊戲：**
- Tower defense 核心玩法
- 玩法與敘事相對分離
- 可以「跳過劇情」或「之後再看」
- 但劇情實際上是賣點之一

**角色設計哲學：**
- 每個角色都有完整的背景設定
- 即使是低稀有度角色
- 「信賴」系統解鎖角色故事
- 角色不只是數值，是「人」

**中國原創的意義：**
- 證明「VN 式敘事」不只是日本特產
- 建立了中國重敘事手遊的模式
- 繁體中文、日文、英文同步
- 全球化的敘事野心

### 原神（Genshin Impact，2020-）

**與 VN 的距離：**
- 開放世界 + 動作 RPG 核心
- 敘事佔比較低（相對於 FGO、方舟）
- 但對話系統仍有 VN 影響

**角色敘事的呈現：**
- 主線任務的對話場景
- 角色傳說任務（專屬劇情）
- 角色突破所需的「角色故事」
- 生日郵件等擬社會關係設計

**全球化成功帶來的影響：**
- 向非日本/中國玩家展示「VN 式角色消費」
- 抽卡 + 角色收集模式的普及
- 但也稀釋了「敘事密度」的期待

### 碧藍航線（Azur Lane，2017-）

**輕敘事、重角色：**
- 主線劇情存在但非核心賣點
- 角色收集與「後宮」感
- 大量角色，但單角色敘事深度有限
- 「誓約」系統（與角色結婚）

**VN 元素的萃取：**
- 保留了「角色好感度」「親密事件」
- 立繪、對話框、語音
- 但缺乏 VN 的敘事密度
- 更接近「角色收集」而非「閱讀體驗」

### 戀愛手遊類（乙女向/女性向）

**代表作品：**
- 《戀與製作人》（中國，2017）
- 《あんさんぶるスターズ!!》（日本）
- 《ツイステッドワンダーランド》（日本）
- 《光與夜之戀》（中國，2021）
- 《未定事件簿》（中國，2020）

**與傳統乙女 VN 的關係：**
- 核心受眾重疊
- 互動模式相似（與角色對話、增加好感度）
- 但商業模式完全不同
- 從「買斷一個作品」到「持續消費角色」

**敘事結構的適應：**
- 主線劇情 + 角色個人線
- 卡片 = 角色的不同面向
- 活動劇情 = 限時故事
- 「與角色的關係」無限延長

**消費心理的差異：**
- 傳統乙女遊戲：完成攻略，獲得結局
- 手遊乙女：「關係」沒有終點，持續維護
- 類似真實戀愛的開放式結構
- 但也意味著沒有「滿足的結束」

## 敘事結構的變異

### 長期營運 vs 一次性敘事

**傳統 VN 的敘事完整性：**
- 作品有明確的開始與結束
- 伏筆會被回收
- 結局提供閉合感
- 主題有完整的表達

**營運型遊戲的敘事挑戰：**
- 故事不能「結束」（結束 = 遊戲結束）
- 需要持續製造新衝突
- 角色不能真正「死亡」或「離開」（除非是敘事高潮）
- 主題難以收束

**常見的妥協策略：**
- 分部/分季制（如 FGO 的第一部、第二部）
- 給予階段性結局，但保留延續空間
- 「這個故事告一段落，但世界還在繼續」
- 部分成功，但敘事張力難以持續

### 碎片化敘事

**敘事的切割：**
- 主線劇情
- 角色個人劇情（幕間、傳說任務）
- 活動限定劇情
- 世界觀設定（圖鑑、資料）
- 周邊媒體（漫畫、小說、廣播劇）

**玩家體驗的碎片化：**
- 無法「一口氣讀完」
- 體驗順序因人而異
- 錯過限時活動 = 敘事缺失
- 重新理解需要大量社群知識

**設計意圖：**
- 持續提供新內容
- 每次登入都有新東西
- 但犧牲了敘事的完整體驗

### 敘事與玩法的張力

**理想狀況：**
- 玩法強化敘事（打這場戰鬥「感覺」像劇情中的戰鬥）
- 敘事給予玩法意義（為什麼要打這關）
- 兩者相輔相成

**實際狀況：**
- 「跳過劇情」按鈕的存在
- 玩法玩家不關心劇情
- 劇情玩家覺得戰鬥是障礙
- 「自動戰鬥」「掃蕩」功能的必要性

**FGO 的案例：**
- 戰鬥系統本身有一定深度
- 但主線推進時，玩家主要想看劇情
- 困難戰鬥造成「劇情斷點」
- 「打不過這關就看不了後面」的frustration

### 敘事作為「獎勵」

**傳統 VN 的敘事獲取：**
- 付費購買 = 獲得全部敘事的權利
- 玩家的「努力」是閱讀時間本身
- 敘事是目的，不是獎勵

**手遊的敘事獲取：**
- 部分敘事需要「解鎖」（角色信賴、突破材料）
- 部分敘事需要「抽到角色」
- 部分敘事是「限時活動」
- 敘事成為遊戲系統的一部分

**這意味著什麼：**
- 敘事獲取被遊戲化
- 「讀故事」變成「玩遊戲的獎勵」
- 創造持續參與的動力
- 但也製造了 FOMO（錯過恐懼）

## 角色消費模式的變異

### 從「攻略」到「收集」

**傳統 VN（戀愛遊戲）的角色消費：**
- 選擇路線 = 選擇與誰發展關係
- 一周目通常只能選一人
- 多周目體驗不同路線
- 「本命」vs「攻略所有人」

**手遊的角色消費：**
- 角色是可收集的資產
- 可以「同時擁有」所有角色（理論上）
- 「全角色收集」是遊玩目標之一
- 不需要選擇 = 不需要放棄

**心理差異：**
- 傳統 VN：選擇的重量、放棄的遺憾
- 手遊：渴望擁有、未擁有的遺憾
- 從「敘事中的選擇」到「現實中的消費選擇」

### 「擁有」的意義

**傳統 VN 中「擁有」角色：**
- 購買遊戲 = 擁有所有角色的故事
- 完成路線 = 與角色「在一起」
- 存檔作為「關係」的保存

**手遊中「擁有」角色：**
- 抽到 = 獲得使用權
- 但需要「養成」才能完整體驗
- 限定版本 = 不同的「他/她」
- 服裝/造型 = 更多的「擁有」維度

**付費「購買」的差異：**
- 傳統 VN：確定性（付這筆錢 = 獲得這些內容）
- 手遊抽卡：概率性（付這筆錢 = 獲得抽的機會）
- 「天井」機制作為確定性的妥協
- 消費金額上限的消失

### 角色的「版本」問題

**傳統 VN：**
- 一個角色就是一個角色
- 可能有不同路線的不同結局
- 但「她」是同一個人

**手遊的角色版本：**
- 同一角色可能有多個卡片/版本
- 不同稀有度、不同造型、不同時期
- 「夏日版」「新年版」「周年版」
- 收集所有版本 = 「完整」擁有這個角色

**這創造了什麼：**
- 對同一角色的重複消費
- 「限定」版本的稀缺性焦慮
- 角色設計的商品化考量
- 「泳裝回」「浴衣回」的必然性

## 商業模式對創作的影響

### 角色設計的商業考量

**「能賣」的角色：**
- 視覺吸引力優先
- 人氣屬性的堆疊
- 「抽這個角色」的慾望製造
- 敘事深度可能為次要考量

**「人氣角色」的持續供應：**
- 需要定期推出新角色
- 舊角色的「新版本」
- 角色數量膨脹
- 難以對每個角色給予相同的敘事投資

**設計的自我審查：**
- 避免讓角色「退場」（會影響商品價值）
- 角色關係不能太封閉（妨礙玩家投射）
- 「這個角色會不會影響其他角色的銷售」
- 商業考量滲透敘事決策

### 活動/限時內容的影響

**為什麼需要限時內容：**
- 創造登入動機
- 製造稀缺性
- 持續提供「新鮮感」
- 營收的週期性高峰

**限時對敘事的影響：**
- 敘事節奏被「活動日程」支配
- 重要劇情可能放在限時活動
- 錯過 = 敘事體驗的永久缺失
- 「復刻」作為部分解決方案

**「限定角色」的焦慮製造：**
- 這個角色只有這次能抽
- 「現在不抽以後就沒有了」
- FOMO 驅動的消費
- 與傳統 VN 的「作品永遠在那裡」完全相反

### 營收壓力與敘事品質

**成功的正循環：**
- 高品質敘事 → 玩家投入 → 願意消費 → 更多資源 → 更高品質
- FGO 第六章後的品質提升
- 「證明自己」後獲得更多創作自由

**失敗的負循環：**
- 敘事投資不足 → 玩家不投入 → 營收不佳 → 更少資源 → 品質更低
- 許多手遊的快速停服
- 「先把遊戲做出來再說」的心態

**中間地帶的掙扎：**
- 維持營運但缺乏突破
- 「夠好」但不「卓越」
- 持續產出但敘事能量遞減
- 長期營運的創作疲勞

## 玩家社群的分化

### 「遊戲玩家」vs「故事讀者」

**遊戲玩家優先：**
- 關注數值、配隊、攻略
- 可能跳過劇情
- 「Meta」討論
- 角色是「強度」

**故事讀者優先：**
- 關注劇情、角色、世界觀
- 可能對戰鬥系統興趣有限
- 「這段劇情好感動」
- 角色是「人」

**重疊區：**
- 兩者都關注的玩家
- 「又強又有故事」的角色最受歡迎
- 但設計上難以兼顧

### 消費能力的分層

**重課玩家（鯨魚）：**
- 角色全收集
- 敘事全解鎖
- 可能是最完整的敘事體驗者
- 但消費金額可達數萬美金

**輕課/月卡玩家：**
- 選擇性收集
- 部分敘事缺失
- 需要取捨

**無課玩家：**
- 只能獲得有限角色
- 大量敘事無法體驗
- 「不付錢就不配看完整故事」的隱含訊息

**這與傳統 VN 的對比：**
- 傳統 VN：付一筆錢，所有人體驗相同
- 手遊：付多少錢，決定體驗多少
- 敘事體驗的階級化

### 社群討論的特徵

**與傳統 VN 社群的差異：**
- 手遊社群更龐大、更活躍
- 但討論更分散（抽卡、強度、劇情、同人）
- 「劇透」問題因進度不同而複雜
- 「還沒抽到所以不能看角色故事」的狀況

**迷因與二創的爆發：**
- 角色數量多 → 更多創作素材
- 持續更新 → 持續有新話題
- 社群創作反過來影響官方
- 「這角色在二創裡的形象」

## 對傳統 VN 市場的影響

### 玩家時間的競爭

**有限的娛樂時間：**
- 手遊設計為「每日遊玩」
- 佔據碎片時間
- 傳統 VN 需要「坐下來認真閱讀」
- 兩者在爭奪同一群玩家

**閱讀習慣的改變：**
- 手遊訓練了「短篇、碎片化」的閱讀
- 傳統 VN 的「長篇沉浸」可能變得困難
- 耐心的下降
- 「太長了，不想讀」

### 敘事期待的改變

**手遊帶來的期待：**
- 定期更新
- 新角色、新故事
- 持續互動
- 「這遊戲還在營運」= 還有新內容

**傳統 VN 的「缺點」：**
- 一次性體驗
- 結束就結束了
- 沒有後續更新
- 在習慣手遊後可能感到「不夠」

### 市場的重組

**誰在玩傳統 VN：**
- 老玩家的堅持
- 對「完整敘事」有追求的玩家
- 討厭抽卡/付費模式的玩家
- 追求「經典」的玩家

**誰轉向了手遊：**
- 對便利性有要求
- 喜歡「持續有新東西」
- 能接受概率付費
- 社群互動的需求

**新玩家的入口：**
- 手遊作為「第一次接觸 VN 式敘事」
- 但可能永遠不會接觸傳統 VN
- 「VN 是什麼？」vs「FGO 我知道」
- 傳統 VN 成為小眾中的小眾

## 創作者的流動

### 從傳統 VN 到手遊

**奈須きのこ與 TYPE-MOON：**
- 從同人 VN 到商業 VN 到手遊帝國
- FGO 的成功讓 TYPE-MOON 成為大型企業
- 但也改變了創作的節奏與方式

**其他案例：**
- Key 社與手遊的嘗試（《Heaven Burns Red》）
- 傳統 VN 公司開發手遊的模式
- 成功與失敗並存

### 手遊原創的脚本家

**新生代的創作者：**
- 直接在手遊框架內創作
- 可能沒有傳統 VN 創作經驗
- 適應了手遊的敘事節奏
- 不同的「什麼是好敘事」標準

**技能的差異：**
- 長篇完整敘事 vs 碎片化敘事
- 明確結局 vs 開放延續
- 單機體驗 vs 社群互動
- 不同的核心能力

### IP 的跨媒體流動

**手遊→其他媒體：**
- 動畫化（FGO、方舟、原神）
- 漫畫化
- 小說化
- 舞台劇

**敘事的「補完」：**
- 遊戲敘事不完整，靠其他媒體補充
- 或者其他媒體的「平行故事」
- 玩家需要追蹤多個媒體才能「完整」
- 跨媒體敘事的碎片化加劇
