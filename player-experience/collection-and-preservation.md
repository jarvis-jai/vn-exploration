# VN 收藏、典藏與保存文化

視覺小說圍繞著一種獨特的收藏文化。這不只是「買遊戲」那麼簡單——VN 玩家發展出一套複雜的收藏實踐、價值判斷與保存焦慮。這份文件探索這個現象的多個面向：實體收藏、數位所有權、社群保存努力、以及「擁有」一部 VN 意味著什麼。

---

## 實體收藏的世界

### 日本市場的實體文化

**盒裝版的標準配置（商業作品）：**

| 項目 | 說明 |
|------|------|
| 遊戲本體 | 光碟或 DVD |
| 說明書 | 基本操作、角色介紹 |
| 特典冊子 | 設定資料、短篇小說、製作者訪談 |
| 回函明信片 | 用戶登錄、問卷調查 |
| 宣傳單 | 同公司其他作品 |

**限定版的額外內容：**
- 原畫設定集
- 原聲 CD
- 掛軸、海報
- 抱枕套（特定作品）
- 角色立牌
- 特典短篇 Drama CD
- 電話卡（昭和-平成時代的遺物）

**為什麼實體版仍有市場：**
- 「擁有感」的實體化
- 收藏展示的滿足
- 限定內容的稀缺性
- 對數位服務停止的擔憂
- 二手市場的流通性

### 西方市場的實體版

**Kickstarter / Limited Run Games 模式：**
- 眾籌達標後才生產
- 數量有限，錯過不再
- 價格通常較高
- 物流等待時間長

**常見配置：**
- 標準版：遊戲本體、說明書
- 收藏版：加 OST、設定集
- 限定版：加抱枕套、海報等

**與日本市場的差異：**
- 實體版的主流程度較低
- 收藏者更多是「硬核粉絲」
- Steam 數位版是默認選擇
- 主機版（Switch 實體）是新興趨勢

### 中文市場的實體情況

**早期盜版時代：**
- 正版實體難以取得
- 盜版光碟流通
- 「正版」概念較薄弱

**現代轉型：**
- 國產 VN 開始推實體限定版
- 代理進口的日本作品有實體選項
- 但數位化程度已高

**臺灣市場：**
- 代理商引進日本作品
- 中文化實體版有收藏市場
- 限定版在預購階段常售罄

## 數位「所有權」的問題

### 你真的「擁有」你買的 VN 嗎？

**Steam / DLsite 的使用條款：**
- 購買的是「使用授權」而非「所有權」
- 平台保留撤銷授權的權利
- 離線遊玩可能受限
- 帳號被封 = 失去所有遊戲

**DRM 的影響：**
- 部分 VN 有 DRM 保護
- 平台關閉後可能無法遊玩
- DRM-free 成為部分玩家的選擇標準

**GOG 與 DRM-free 的吸引力：**
- GOG 主打無 DRM
- 下載後可永久保存
- VN 玩家中有一群人特別重視這點
- 但 GOG 的 VN 庫相對有限

### 消失的遊戲

**已無法合法取得的 VN：**
- 公司倒閉、版權不明
- 從所有平台下架
- 僅存於二手市場或灰色地帶

**具體情況類型：**

1. **公司倒閉**
   - 遊戲不再銷售
   - 網站關閉
   - 版權歸屬不清

2. **授權到期**
   - 合作開發的作品
   - 授權方不再續約
   - 或雙方有爭議

3. **主動下架**
   - 公司決定不再銷售
   - 可能因為「品牌形象」
   - 或作品「過時」不想維護

4. **平台關閉**
   - 特定平台獨佔的作品
   - 平台關閉後無法取得
   - 主機數位商店的風險

**玩家的因應：**
- 備份已購買的遊戲
- 尋找二手實體版
- 進入灰色地帶（盜版、破解）

### 「擁有」vs「體驗」的哲學

**收藏者心態：**
- 「我要能隨時再玩」
- 「我要確保這個作品不會消失」
- 「擁有本身就是價值」

**體驗者心態：**
- 「玩過就好，不需要留著」
- 「反正也不會重玩」
- 「訂閱制也可以接受」

**兩種態度的張力：**
- 收藏者認為體驗者「不尊重作品」
- 體驗者認為收藏者「囤積癖」
- 但兩者可能是同一人在不同作品上的態度

## 遊戲保存運動

### 為什麼 VN 需要被保存

**VN 特別脆弱的原因：**
- 許多小公司、同人作品
- 技術依賴過時的系統
- 市場小，保存意識低
- 語言障礙限制國際保存

**時間的威脅：**
- 老舊引擎無法在新系統運行
- 光碟劣化
- 網站關閉、下載連結失效
- 製作者去世、公司解散

### 社群保存的努力

**VNDB 的角色：**
- 記錄存在過的所有 VN
- 即使已無法取得，仍有記錄
- 元資料保存（作者、發售日、角色等）
- 但不提供遊戲本體

**Internet Archive 與 VN：**
- 部分老舊 VN 被上傳
- 法律灰色地帶
- 但對保存有實際貢獻

**私人收藏者：**
- 有人專門收集絕版作品
- 不一定公開分享
- 但確保「某處有一份」

**翻譯組的附帶貢獻：**
- 翻譯過程會保存遊戲文本
- 破解 DRM 以應用翻譯
- 無意間成為保存力量

### 商業與保存的衝突

**公司的立場：**
- 老舊作品沒有商業價值
- 維護成本高於收益
- 可能與品牌形象衝突
- 「就讓它消失吧」

**玩家的訴求：**
- 「至少讓我們買」
- 「開放原始碼或公有領域」
- 「不要阻擋民間保存」

**現實的妥協：**
- 極少數公司會主動保存
- 有時會出復刻版（但可能改動）
- 大多數老作品的命運是消失

### 技術保存的挑戰

**引擎相容性：**
- 老舊 VN 使用特定引擎
- Windows XP 時代的程式
- 可能無法在現代系統運行

**解決方案：**
- 虛擬機器模擬
- Wine / Proton 相容層
- 原生移植（需要原始碼）

**粉絲移植的案例：**
- 部分有技術能力的粉絲
- 將老遊戲移植到現代系統
- 法律地位不明但貢獻明確

## 收藏者的類型學

### 完整主義者（Completionist）

**特徵：**
- 要收集某作品或某公司的「所有東西」
- 不只遊戲，還有周邊、設定集、CD
- 對「版本差異」敏感
- 可能追求「未開封品」

**收藏範圍：**
- 初回限定版
- 通常版
- 廉價版
- 各主機移植版
- 相關書籍、CD、周邊

**驅動力：**
- 完成感
- 對作品的極度喜愛
- 收藏本身的滿足

### 精選主義者（Curator）

**特徵：**
- 只收「值得」的作品
- 有自己的篩選標準
- 可能專注某類型或某公司
- 品質優先於數量

**收藏邏輯：**
- 「這是該類型的代表作」
- 「這個版本的特典最好」
- 「這家公司的作品我都收」

### 投機者（Speculator）

**特徵：**
- 預測什麼會升值
- 囤積限定版等待漲價
- 密切關注二手市場行情
- 可能不是為了自己玩

**行為：**
- 買多套限定版
- 保持未開封
- 在適當時機出售

### 保存者（Archivist）

**特徵：**
- 使命感驅動
- 「這個作品不能消失」
- 可能收集自己不會玩的作品
- 關注瀕危、絕版、冷門作品

**行為：**
- 備份數位版
- 收集老舊作品
- 參與社群保存計畫
- 分享或不分享（取決於個人立場）

## 二手市場的生態

### 日本的二手 VN 市場

**主要管道：**
- 駿河屋（Suruga-ya）
- Mandarake
- Amazon Japan Marketplace
- Yahoo! 拍賣
- Mercari

**價格規律觀察：**
- 限定版通常高於通常版
- 絕版 = 潛在高價
- 冷門作品可能便宜也可能貴
- 狀態（開封/未開封）影響價格

**跨境購買的障礙：**
- 許多店不寄國外
- 需要轉運服務
- 運費昂貴
- 但對收藏者而言值得

### 西方市場的二手流通

**主要管道：**
- eBay
- Amazon Marketplace
- 專門的收藏品網站
- Reddit / Discord 私下交易

**與日本的差異：**
- 流通量較小
- 價格波動較大
- 實體版本來就少

### 價值的建構

**什麼讓一個 VN 「有價值」：**

| 因素 | 影響 |
|------|------|
| 稀有度 | 限定版、絕版 |
| 需求 | 人氣作品、經典 |
| 狀態 | 未開封、完整 |
| 版本 | 初版、首刷 |
| 特典 | 附帶特典完整 |
| 簽名 | 製作者、聲優簽名 |

**價值與「遊玩價值」的脫鉤：**
- 有些高價作品不是因為好玩
- 而是因為稀有
- 收藏市場有自己的邏輯

## 數位收藏的實踐

### 硬碟上的圖書館

**數位收藏者的習慣：**
- 維護整理過的遊戲資料夾
- 備份到多個硬碟
- 有時備份到雲端（但有隱私顧慮）
- 保留安裝檔、補丁、存檔

**資料夾組織美學：**
- 按公司
- 按年份
- 按類型
- 或複合分類

**元資料管理：**
- 有些人會維護試算表
- 記錄遊玩狀態、評分、筆記
- VNDB 的「遊玩列表」功能

### 存檔的珍貴

**存檔作為「記憶」：**
- 存檔不只是進度
- 是「那次遊玩」的證明
- 有些人會保留多年的存檔

**存檔的危險：**
- 系統重灌可能遺失
- 存檔格式可能不相容新版
- Cloud Save 的可靠性？

**存檔的社群分享：**
- 有些人分享 100% 完成的存檔
- 讓其他人直接解鎖所有 CG
- 也有人認為這「作弊」

### 截圖與紀錄

**遊玩中的截圖習慣：**
- 紀錄喜歡的場景
- 保存 CG（即使遊戲有回想）
- 記錄「這裡讓我哭了」的時刻

**截圖的整理：**
- 有人會按作品建資料夾
- 有人會做成桌布
- 有人會分享到社群

**影片錄製：**
- 較少見但存在
- 主要是實況主
- 或想記錄自己的反應

## 「玩過」的證明

### 數位時代的「完成」

**怎麼證明你「玩過」一個 VN：**
- 成就系統（如 Steam 成就）
- CG 全回收
- 全路線完成
- 遊玩時間記錄

**但這些都可以偽造：**
- 成就可以解鎖器
- 存檔可以下載
- 時間可以掛機

**「證明」的意義：**
- 主要是對自己
- 社群中有時會問「你真的玩完了嗎」
- 但通常不會深究

### 實體收藏作為證明

**展示收藏：**
- 書架上的遊戲盒
- 社群媒體的收藏照片
- 「看，我真的買了」

**收藏 ≠ 遊玩：**
- 有人收藏了但沒玩
- 「積灰」的遊戲庫
- 這本身也是一種收藏者現象

### 「背德感」的缺席

**VN 收藏的特殊氛圍：**
- 因為市場小眾
- 「支持作品」的正當性
- 買了但沒玩比較不會被批評
- 「至少我付錢了」

## 復刻、重製與收藏

### 復刻版的意義

**對收藏者的影響：**
- 讓更多人能玩到
- 但可能稀釋原版價值
- 也可能提高原版稀有度認知

**版本差異的問題：**
- 復刻版可能有修改
- 音樂授權問題導致替換
- 圖像重繪
- 「哪個版本才是正典」

### 重製版的取捨

**完全重製：**
- 新引擎、新立繪
- 可能被視為「另一個作品」
- 收藏者可能兩個都要

**畫質提升版（Remaster）：**
- 提升解析度
- 修 Bug
- 較不會被視為「不同作品」

**收藏者的態度光譜：**
- 「只收原版」
- 「只收最新版」
- 「全版本收」

## 收藏與身份認同

### 「我的收藏代表我」

**收藏作為自我表達：**
- 「我是 Key 社粉」
- 「我專攻世界系」
- 「我收冷門作品」

**展示收藏的社交功能：**
- Twitter / 社群的收藏照片
- 引發討論與連結
- 找到同好

### 收藏的隱私

**成人向作品的收藏：**
- 不會公開展示
- 但對收藏者同樣有意義
- 「隱藏的書架」

**與非玩家的落差：**
- 家人、朋友可能不理解
- 收藏可能需要「解釋」
- 或選擇不解釋

## 消費倫理的考量

### 「支持創作者」的論述

**購買正版的正當性建構：**
- 「這樣才能有續作」
- 「尊重創作者的勞動」
- 「維持產業的運作」

**但複雜的現實：**
- 公司倒閉後購買無法支持創作者
- 二手購買錢不會到創作者手中
- 數位平台抽成的問題

### 盜版的灰色地帶

**常見的正當化論述：**
- 「我所在地區無法購買」
- 「我買不起但想玩」
- 「玩過之後會補票」
- 「這個作品已經無法購買了」

**社群的態度：**
- 大多數社群表面上反對盜版
- 但私下討論或暗示
- 「保存」與「盜版」的界線模糊

### 「消費即參與」

**購買行為的意義超過金錢：**
- 市場數據影響產業決策
- 「用錢投票」
- 購買也是表態

## 收藏的終點

### 收藏者的老去

**當收藏者不再收藏：**
- 興趣轉移
- 空間限制
- 經濟考量
- 生命階段變化

**收藏的歸宿：**
- 出售
- 贈送
- 丟棄
- 遺贈

### 「這些東西會怎樣？」

**收藏者的焦慮：**
- 「我死後這些怎麼辦」
- 「家人會不會當垃圾丟」
- 「有沒有人接手」

**可能的解法：**
- 找到同好接手
- 捐給相關機構（但很少有這種機構）
- 接受「會消失」的現實
