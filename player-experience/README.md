## 體驗分類

### 依閱讀密度

| 類型 | 特徵 |
|------|------|
| 文字密集 | 大量敘述，接近小說 |
| 對話驅動 | 以對話為主，節奏快 |
| 視覺優先 | 文字精簡，重美術 |

### 依互動程度

| 類型 | 特徵 |
|------|------|
| Kinetic Novel | 無選擇，純閱讀 |
| 選擇輕量 | 偶爾選項 |
| 選擇密集 | 頻繁選擇 |
| 混合型 | 含解謎/養成 |

### 依情感

- 治癒 / 放鬆
- 緊張 / 懸疑
- 催淚 / 感動
- 恐怖 / 不安
- 燃 / 熱血
- 致鬱 / 沉重
