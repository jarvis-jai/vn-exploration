# 常見失敗模式（Failure Modes）

VN 開發中常見的問題與陷阱。本目錄整理這些失敗模式，作為開發時的檢查清單。

---

## 這裡收錄什麼

- 敘事設計常見問題
- 節奏控制失誤
- 選擇設計陷阱
- 技術與呈現問題
- 商業與發行錯誤

---

## 失敗模式清單（待填充）

### 敘事問題

- **開場過長**
  - 進入主要劇情前的冗長介紹
  - 玩家在有趣的事發生前就離開

- **角色塑造薄弱**
  - 角色缺乏動機或一致性
  - 對話千篇一律

- **結局不滿足**
  - 伏筆未回收
  - 結局過於突兀或敷衍

- **主題不明確**
  - 故事想說什麼不清楚
  - 多個主題互相衝突

### 節奏問題

- **中段拖沓**
  - 缺乏中段高潮
  - 玩家失去動力

- **節奏不一致**
  - 某些段落過長或過短
  - 緊張與舒緩的轉換生硬

### 選擇設計問題

- **假選擇過多**
  - 選項不影響任何事
  - 玩家感覺被欺騙

- **選擇後果不明**
  - 玩家不知道選擇會造成什麼影響
  - 導致隨意選擇或存檔重讀

- **正確答案過於明顯**
  - 選項設計缺乏道德灰色地帶
  - 沒有真正的抉擇感

### 技術與呈現問題

- **文字排版問題**
  - 字型大小不適當
  - 行距或字距不佳

- **音效 / BGM 使用不當**
  - 音量平衡問題
  - 音樂與場景不搭

- **UI / UX 不直覺**
  - 存檔系統難用
  - 跳過 / 快進功能缺失

### 商業問題

- **定價與內容不符**
  - 價格過高但內容過短
  - 缺乏價值感知

- **發行時機問題**
  - 上架後缺乏維護
  - 首發 bug 過多
