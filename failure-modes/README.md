## 失敗模式速查

### 敘事問題

| 問題 | 症狀 |
|------|------|
| 開場過長 | 玩家在有趣的事發生前離開 |
| 角色薄弱 | 缺乏動機，對話千篇一律 |
| 結局不滿足 | 伏筆未回收，結局突兀 |
| 主題不明 | 故事想說什麼不清楚 |

### 節奏問題

| 問題 | 症狀 |
|------|------|
| 中段拖沓 | 缺乏中段高潮 |
| 節奏不一 | 緊張與舒緩轉換生硬 |

### 選擇設計

| 問題 | 症狀 |
|------|------|
| 假選擇過多 | 選項不影響任何事 |
| 後果不明 | 玩家不知道選擇會怎樣 |
| 答案太明顯 | 沒有真正的抉擇感 |

### 技術問題

| 問題 | 症狀 |
|------|------|
| 排版差 | 字型/行距不適當 |
| 音效失調 | 音量問題，配樂不搭 |
| UI 不直覺 | 存檔難用，快進缺失 |

### 商業問題

| 問題 | 症狀 |
|------|------|
| 定價錯誤 | 價格與內容不符 |
| 發行問題 | 首發 bug，缺乏維護 |
