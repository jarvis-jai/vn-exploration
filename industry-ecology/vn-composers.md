# VN 作曲家文化：聲音的作者性

如同脚本家能在 VN 界獲得個人認知度，作曲家同樣是少數能讓玩家追隨其名字的創作者類型。「這首曲子是誰寫的」在 VN 社群中是有意義的問題——甚至可能決定玩家是否購買某部作品。

---

## 為什麼 VN 作曲家能建立品牌

### 與其他遊戲音樂的差異

**聆聽時間極長：**
- 單一 BGM 可能被聽數百次
- 玩家與音樂建立深度記憶連結
- 不像動作遊戲——音樂是「背景雜音」
- VN 的靜態畫面讓音樂更突出

**情感記憶的錨定：**
- 「一聽到這首曲子就想哭」
- 音樂成為重要場景的代名詞
- Vocal 曲目尤其如此
- 作曲家成為「那個情感體驗」的署名

**有限的核心創作者圈：**
- VN 產業不大
- 優秀作曲家數量有限
- 同一批人反覆出現在名作中
- 久而久之建立辨識度

**音樂商品化：**
- VN OST 是獨立商品
- 音樂會、演唱會的舉辦
- 作曲家有機會站到台前
- 與粉絲直接接觸

## 代表性作曲家側寫

### 折戸伸治（Shinji Orito）

**所屬：** Key

**代表作品：**
- 《Kanon》
- 《AIR》
- 《CLANNAD》
- 《Little Busters!》
- 《Rewrite》

**風格特徵：**
- 鋼琴主導的抒情旋律
- 「Key 之音」的核心建立者之一
- 溫暖而帶有淡淡憂傷的基調
- 善用簡單和弦進行營造情感
- 日常曲明快但不輕浮

**經典曲目：**
- 「鳥の詩」（AIR OP）- 與 Lia 的合作
- 「夏影」（AIR）
- 「渚」（CLANNAD）
- 「潮鳴り」（CLANNAD）
- 「Little Busters!」（同名 OP）

**產業地位：**
- 「泣きゲー音樂」的定義者之一
- Key 作品的聲音身份
- 影響了整個催淚系 VN 的音樂美學

## 音樂製作公司與圈子

### Key / Visual Art's 內部

**特徵：**
- 極高的作曲家忠誠度
- 多人長期合作形成「Key 之音」
- 折戸伸治、戸越まごめ、麻枝准的核心三角
- 後期加入的作曲家也維持風格一致性

### I've Sound

**全盛期：** 2000-2010 年代初

**特徵：**
- 大量 VN 主題曲的供應商
- 培養了多位知名歌手
- KOTOKO、川田まみ、島みやえい子
- 電子流行風格的代名詞
- 逐漸轉型或淡出 VN 市場

**影響：**
- 定義了 2000 年代 VN OP 的聲音
- 「エロゲソング」的主流美學

### Elements Garden

**特徵：**
- 上松範康主導
- 大量參與音樂遊戲與媒體混合企劃
- 偶爾參與 VN 音樂製作

### 獨立 / 同人音樂社團

**例如：**
- IOSYS（音MAD 與東方系，偶爾參與 VN）
- 各種 M3 販售的音樂社團
- 為同人 VN 提供音樂

## 作曲家的跨界與流動

### VN → 動畫

**常見路徑：**
- VN 改編動畫時，原作作曲家參與
- 或動畫另請作曲，但保留經典曲目
- 作曲家藉此獲得更大曝光

**例子：**
- Key 作品動畫化時的音樂處理
- 《Fate/stay night》動畫的音樂策略

### VN → 主機遊戲

- 部分作曲家同時參與 VN 與其他遊戲
- 技術要求不同（互動音樂 vs 線性）
- 但基本美學可轉移

### 商業 ↔ 同人

- 同人出身的作曲家進入商業
- 商業作曲家參與同人企劃
- 兩個圈子的人脈流通

## 音樂授權與版權

### OST 發售

**常見形態：**
- 特典 CD（限定版附帶）
- 單獨發售的 OST 專輯
- 數位發行（近年增加）

**版權歸屬：**
- 通常歸製作公司所有
- 作曲家有署名權但不一定有重製權
- 影響二次創作（cover、翻唱）

### 演唱會與現場

**Key 20 周年演唱會等：**
- 作曲家登台
- 粉絲與創作者的直接接觸
- 經典曲目的管弦樂團演奏

**意義：**
- VN 音樂被視為「值得現場演奏」的藝術
- 提升作曲家的社會地位
- 粉絲社群的凝聚

## 音樂風格的時代變遷

### 1990 年代：MIDI 限制期

**特徵：**
- 硬體限制下的作曲
- FM 音源、簡單波形
- 音色選擇受限
- 但仍有創意突破

### 2000 年代初：CD-DA 普及

**特徵：**
- 可以使用實際錄音
- 音質大幅提升
- 開始出現管弦樂編曲
- I've Sound 風格的電子流行

### 2000 年代後期：HD 時代

**特徵：**
- 高品質採樣音源普及
- 小型團隊也能製作高品質管弦樂
- 同人與商業品質差距縮小

### 2010 年代至今

**特徵：**
- 數位發行讓音樂更易取得
- Streaming 對 OST 銷售的衝擊
- 獨立 VN 的音樂來源多樣化（授權音樂庫等）

## 玩家對作曲家的認知模式

### 「追作曲家」的玩家

- 「有折戸的曲子就買」
- 對特定作曲家的信任
- 類似追脚本家的行為

### 音樂作為購買考量

- OST 試聽影響購買決定
- 「音樂好聽」作為推薦理由
- OP 曲的重要性

### 社群討論

- VNDB 有音樂 staff 標註
- 討論串會特別讚美或批評音樂
- 「這部的 BGM 是弱點」作為批評
