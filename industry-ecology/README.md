視覺小說的製作與發行生態，從同人社團到商業公司，從日本國內到全球市場。這個目錄探索「誰在做 VN」「如何運作」「生態如何演變」。

## 本目錄範圍

- 開發者類型與規模譜系
- 同人（Doujin）vs 商業生態
- 知名製作公司的風格與歷史
- 獨立開發者（西方 Indie）的興起
- 發行商與平台的角色
- 開發團隊的組成結構
- 產業的經濟循環
