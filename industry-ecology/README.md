# 產業生態（Industry Ecology）

誰在做 VN、如何運作、生態如何演變。

本目錄聚焦於 VN 產業的「人」與「組織」——工作室的興衰、創作者的職涯路徑、商業模式的演變、以及各地區生態的差異。

---

## 這裡收錄什麼

- 日本商業 VN 公司與同人圈的運作模式
- 腳本家、作曲家、繪師等核心創作者的職涯生態
- 西方獨立 VN 場景的發展脈絡
- 翻譯發行商與在地化產業鏈
- 眾籌、Patreon 等新興資助模式
- 工作室倒閉與產業結構變化的觀察

## 快速參考：日本 VN 工作室類型

| 類型 | 特徵 | 代表 |
|------|------|------|
| 大型商業 | 穩定發行、多品牌、有資本背景 | Visual Arts（Key）、Nitroplus、AQUAPLUS |
| 中型商業 | 專注特定類型、作品數較少但品質穩定 | August、FAVORITE、ゆずソフト |
| 同人出身 | 從同人圈起家、後商業化 | TYPE-MOON、07th Expansion、Liar-soft |
| 乙女專門 | 女性向戀愛 VN | オトメイト、Rejet、Ruby Party |
| 已解散 | 產業結構變化下的犧牲者 | minori（2019）、feng（2020）、OVERDRIVE（2019） |

## 快速參考：西方 VN 開發者類型

| 類型 | 特徵 | 代表 |
|------|------|------|
| 知名獨立 | 多部作品、建立個人品牌 | Christine Love、ebi-hime、Hanako Games |
| 商業成功 | 銷量突破、被廣泛討論 | Team Salvato（DDLC）、Sukeban Games（VA-11 Hall-A） |
| 小型工作室 | 2-5 人團隊、持續產出 | Winter Wolves、Studio Élan、GB Patch Games |
| 東南亞新興 | 非英語母語但以英語發行 | Toge Productions（印尼）、Woodsy Studio（菲律賓） |
| Game Jam 出身 | 從 NaNoRenO 等 Jam 起步 | 眾多 itch.io 開發者 |
