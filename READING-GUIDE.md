# 閱讀指南 (Reading Guide)

> 給第一次閱讀這個 repo 的人。

---

## 這個 repo 是什麼

一個關於視覺小說（VN）的知識索引。不是教學，不是推薦清單，不是決策文件。只是把「VN 世界裡有什麼」整理成可查閱的形式。

---

## 建議閱讀順序

### 如果你是完全的新手

1. `genres/README.md` — 先看類型總覽
2. `representative-works/` — 瀏覽代表作品，建立直覺
3. `narrative-structures/choice-design.md` — 理解 VN 的核心：選擇設計

### 如果你想了解市場與產業

1. `monetization/` — 商業模式與定價
2. `industry-ecology/doujin-commercial-spectrum.md` — 同人 vs 商業
3. `industry-ecology/platform-evolution.md` — 平台變遷史
4. `communities/` — 各地區社群生態

### 如果你想學習敘事設計

1. `narrative-structures/opening-sequences.md` — 開場設計
2. `narrative-structures/choice-design.md` — 選擇設計
3. `narrative-structures/endings-and-termination.md` — 結局設計
4. `failure-modes/` — 從失敗中學習

### 如果你對特定類型有興趣

直接進 `genres/` 找對應的類型專文：
- 恐怖 → `horror-spectrum.md`
- 推理 → `mystery-suspense.md`
- 治癒 → `iyashikei-healing.md`
- 百合 → `yuri-girls-love.md`
- 乙女 → `otome-games.md`

### 如果你想做開發

1. `tooling/renpy-ecosystem.md` — Ren'Py 是最常見的選擇
2. `failure-modes/production-economics.md` — 開發成本現實
3. `localization/vn-specific-challenges.md` — 在地化挑戰
