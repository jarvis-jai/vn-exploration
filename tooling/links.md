## 引擎選型快速指南

### 按技術能力

| 能力等級 | 推薦引擎 |
|---------|---------|
| 無程式經驗 | 橙光、Visual Novel Maker、TyranoBuilder |
| 基礎程式（Python） | Ren'Py |
| 中等程式（GDScript） | Godot + Dialogic |
| 進階程式（C#） | Unity + Naninovel/Fungus |
| Web 開發背景 | Monogatari |

### 按專案需求

| 需求 | 推薦方向 |
|------|---------|
| 純文字 VN | Ren'Py（最成熟） |
| 日系商業風格 | Ren'Py、Unity + Utage |
| 混合類型（RPG+VN） | Godot、Unity |
| 手機優先 | Unity、Godot |
| 快速原型 | Ren'Py、TyranoBuilder |
| 3D / Live2D 重度使用 | Unity |

### 按發行目標

| 平台 | 引擎支援 |
|------|---------|
| Steam（最廣） | 全部 |
| 主機（Switch等） | Unity > Godot > Ren'Py（需移植） |
| Web / 瀏覽器 | Monogatari、Ren'Py Web、Unity WebGL |
| 手機 | Unity、Godot、Ren'Py（有限制） |
