# VN 保存與數位典藏

視覺小說作為一種數位媒體形式，面臨獨特的保存挑戰。不同於傳統書籍或電影，VN 的存續依賴特定的軟體環境、硬體相容性、以及權利持有者的持續維護意願。本文件探索 VN 保存的現狀、挑戰與社群行動。

---

## 保存危機的規模

### 消失中的作品

**無法再合法取得的作品：**
- 發行商倒閉後下架的作品
- 實體版售罄、數位版未發行的老作品
- DRM 伺服器關閉導致無法啟動的作品
- 平台獨佔作品（如某些手機 VN）在服務終止後消失

**估計規模：**
- VNDB 收錄的作品中，相當比例已無法合法購買
- 1990 年代至 2000 年代初期的 PC-98 / Windows 95/98 時代作品流失嚴重
- 手機 VN（特別是功能型手機時代）幾乎全部消失
- 中國早期橙光作品因平台政策變動而消失

**具體案例觀察：**

*PC-98 時代：*
- 大量早期 VN 僅在 PC-98 平台發行
- 需要模擬器或原機才能運行
- 原版軟碟取得困難且易損壞
- 許多作品從未數位化或重製

*功能型手機時代：*
- 日本的 i-mode / EZweb 平台 VN
- 隨平台關閉而完全消失
- 即使保存了資料，也難以模擬運行環境

*中國特殊情況：*
- 橙光平台的創作者帳號停用
- 政策審查導致的內容刪除
- 缺乏系統性的保存意識

## 技術層面的挑戰

### 軟體相容性

**作業系統更迭：**
- Windows 95/98/Me 時代的作品在現代 Windows 上執行困難
- 16 位元應用程式在 64 位元系統上無法原生執行
- macOS 的 32 位元應用支援已移除
- DirectX / DirectDraw 等舊版 API 的相容性問題

**常見技術問題：**
- 螢幕解析度與全螢幕模式
- 音效卡 / MIDI 相容性
- 日文編碼（Shift-JIS）在非日文系統的顯示
- 防拷保護（光碟驗證、序號驗證）

**應對策略：**
- 虛擬機器（VirtualBox、VMware）
- 相容性模式設定
- 社群製作的補丁
- Wine（在 Linux / macOS 上運行 Windows 軟體）

### 引擎與格式

**封閉格式問題：**
- 許多 VN 使用專有引擎與資源格式
- 腳本、圖像、音訊封裝在專屬檔案中
- 缺乏官方工具時，需逆向工程才能提取

**主要引擎生態：**

| 引擎 | 開放程度 | 保存難度 |
|------|----------|----------|
| Ren'Py | 開源 | 低 |
| KiriKiri | 半開放 | 中 |
| 吉里吉里 Z | 開源 | 低 |
| NScripter | 開源 | 低 |
| BGI / Ethornell | 封閉 | 高 |
| RealLive | 封閉 | 高 |
| CatSystem2 | 封閉 | 高 |
| AGES | 封閉 | 高 |

**社群逆向工程：**
- AnimED 等工具可解包/打包部分引擎
- Crass / GARbro 可提取多種格式的資源
- 但逆向工程的法律地位模糊

### DRM 與線上驗證

**問題類型：**
- 需要連接驗證伺服器才能啟動
- 光碟版的防拷保護與現代光碟機不相容
- 數位發行平台的帳號綁定

**已發生的災難：**
- 某些發行商倒閉後，驗證伺服器關閉
- 合法購買的玩家無法再遊玩
- 有時官方會在關閉前發布移除 DRM 的補丁，但並非總是

**玩家應對：**
- 破解補丁（法律灰色地帶）
- 購買實體版保留
- 倡議 DRM-free 發行

## 保存行動者

### 官方層面

**發行商的保存意願參差：**

*積極案例：*
- MangaGamer 持續發行經典作品的現代化版本
- JAST USA 對老作品進行重製
- 某些日本發行商推出「Memorial Edition」

*消極案例：*
- 許多發行商對舊作毫無興趣
- 版權分散導致重製困難
- 經濟誘因不足（舊作銷量有限）

**平台政策：**
- Steam 上的舊作通常有社群維護
- GOG 以 DRM-free 與經典遊戲聞名
- DMM / DLsite 的數位版有時品質參差

### 社群保存

**論壇與維基：**
- Anime Sharing 等論壇的資源分享（灰色地帶）
- Fuwanovel 的資源整理（已轉型為評論站）
- 各語言社群的本地資源庫

**翻譯補丁作為保存：**
- 翻譯補丁通常包含技術修正
- 讓作品在現代系統上可運行
- 翻譯團隊有時比原發行商更積極維護

**特定社群行動：**

*The Visual Novel Archive：*
- 匿名分散式保存計畫
- 法律地位不明確
- 保存優先於法律顧慮的立場

*VNDB 的角色：*
- 作為元資料庫，記錄作品存在過
- 即使作品本身消失，記錄仍在
- 不直接提供檔案，但連結資源

*Internet Archive：*
- 部分 VN 被上傳
- 法律風險由平台承擔
- 保存品質參差

### 模擬器社群

**PC-98 / FM Towns 模擬：**
- Neko Project 21 等模擬器
- 讓舊時代 VN 可在現代硬體運行
- 需要 BIOS 檔案（取得可能涉及法律問題）

**主機 VN 的模擬：**
- PlayStation / Sega Saturn 上的 VN
- 模擬器成熟度高
- 與 PC 版的版本差異問題

## 法律與倫理的張力

### 著作權問題

**核心張力：**
- 保存需要複製與分發
- 未經授權的複製在多數法域屬侵權
- 「孤兒著作」（Orphan Works）問題——無法聯繫權利人

**各法域差異：**
- 日本對著作權執法嚴格
- 美國有 fair use 抗辯但範圍有限
- 歐洲有特定的文化遺產保存例外

**實務現狀：**
- 發行商通常不追究舊作的分享（成本效益不符）
- 但法律風險仍然存在
- 社群保存在法律陰影下進行

### 倫理爭論

**「盜版還是保存？」：**

*支持保存優先的論點：*
> 「如果官方不保存，誰來保存？等作品消失就來不及了。」
> 「文化遺產不應該因為商業失敗而消失。」
> 「我買過這個遊戲，現在買不到了，為什麼不能下載？」

*反對意見：*
> 「這仍然是著作權侵害，不論理由多麼高尚。」
> 「保存不應該繞過權利人，應該倡議更好的法律。」
> 「有些人用『保存』作為免費取得新遊戲的藉口。」

**中間立場：**
- 只保存確實無法購買的作品
- 優先購買可購買的版本
- 支持官方重製與再發行

## 物質媒體的角色

### 實體收藏文化

**收藏動機：**
- 對作品的情感連結
- 收藏品的物質質感
- 擔心數位版消失的風險
- 限定版的獨特內容

**物質形式：**
- 光碟 / DVD-ROM
- 豪華盒裝版
- 設定集、原畫集
- 周邊商品

**收藏與保存的關聯：**
- 收藏家保留了物質載體
- 但物質媒體也會老化
- CD-R / DVD-R 的壽命有限
- 需要定期轉存

### 數位優先時代

**現況轉變：**
- 越來越多作品僅數位發行
- 實體版變成限定或高價選項
- 某些地區（如中國）幾乎全面數位化

**對保存的影響：**
- 數位版依賴平台存續
- 失去物質備份
- 但也更容易複製保存（若無 DRM）

## 後設資料保存

### 作品資訊

**VNDB 的角色：**
- 即使作品本身消失，VNDB 條目保留了：
  - 作品名稱、發行資訊
  - 製作人員名單
  - 類型標籤、劇情簡介
  - 玩家評分與評論
- 這本身就是一種保存

**待補充的資訊類型：**
- 製作幕後故事
- 開發者訪談
- 當年的評論文章
- 社群討論的存檔

### 周邊文本

**需要保存的周邊內容：**
- 攻略本 / 完全攻略
- 設定資料集
- 官方網站（經常在作品停售後關閉）
- 廣告與宣傳素材

**Wayback Machine 的價值：**
- Internet Archive 保存了許多已關閉的官方網站
- 但並非完整覆蓋
- 動態內容難以保存

## 重製與再發行的意義

### 作為保存手段

**理想的重製：**
- 讓作品在現代系統上原生運行
- 保留原作的風味
- 合法且支持權利人

**問題與限制：**
- 僅商業上可行的作品會被重製
- 冷門作品無人問津
- 有時重製會修改原作（審查、版權素材替換）

### 重製的爭議

**原作 vs 重製版：**
- 部分玩家偏好原版的「原汁原味」
- 重製版可能刪改成人內容
- 音樂版權問題導致替換配樂
- 舊畫風 vs 新畫風的爭論

**案例：**
- 某些重製版被認為「改壞了」
- 反而激發對原版的保存需求

## 未來展望

### 潛在改善

**技術面：**
- 模擬技術持續進步
- 容器化 / 虛擬化更易使用
- AI 輔助的舊遊戲相容性修補

**法律面：**
- 孤兒著作法規的改進
- 文化遺產保存例外的擴大
- 發行商主動放棄過時作品的權利

**商業面：**
- 復古遊戲市場的成長
- 訂閱制經典遊戲服務
- 發行商發現舊作的長尾價值

### 社群能做什麼

**個人層面：**
- 購買並保存你珍視的作品
- 備份你的收藏
- 支持合法的保存行動
- 撰寫評論與記錄

**集體層面：**
- 支持 VNDB 等資料庫
- 參與翻譯/社群維護
- 倡議更好的著作權法
- 建立社群標準與最佳實踐

## 反思

視覺小說保存的核心問題不是「能不能做」，而是「誰來做」以及「在什麼法律框架下做」。目前的現實是：

- 官方沒有足夠誘因保存冷門作品
- 社群有動力但面臨法律風險
- 作品每天都在消失
- 我們的集體記憶正在流失

這不只是懷舊問題。每一部消失的 VN 都是敘事藝術歷史的一部分。今天的冷門可能是明天的經典。而一旦消失，就真的消失了。
