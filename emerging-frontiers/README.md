# emerging-frontiers/

## 範圍

收錄 VN 領域的新興邊界議題——那些正在發生但尚未定型的變化、技術、模式，以及可能重塑整個類型的力量。

此目錄不追求「預測」，而是紀錄當下的觀察、實驗、爭議與猜測。

---

## 可能收錄的主題

- **AI 與 VN 的交會**：AI 生成工具、AI 伴侶服務、互動模式的變革
- **新平台與介面**：VR/AR 敘事、手機原生設計、跨裝置體驗
- **發行模式演進**：訂閱制、免費增值、眾籌後發展
- **社群創作工具**：MOD 文化、UGC 平台、開放式敘事
- **監管與審查**：成人內容政策的變化、地區差異
- **跨媒體融合**：VN 與 VTuber、直播、社群媒體的整合

---

## 收錄原則

- 優先收錄「已經發生」的實例，而非純粹的猜測
- 保留爭議性觀點的多方聲音
- 標註資訊的時效性（這個領域變化快）
- 容忍「可能過時」的風險——這是探索的本質
