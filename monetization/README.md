## 商業模式（待填充）

### 定價模式

- **付費買斷**
  - 常見價格帶
  - 價格與內容長度的關係

- **免費 + 捐款（PWYW）**
  - itch.io 模式
  - 小費文化

- **免費 + DLC**
  - 本篇免費
  - 追加路線或內容收費

- **Early Access**
  - 提前發售未完成版本
  - 風險與注意事項

- **訂閱 / Patreon**
  - 章節式發行
  - 持續更新模式

### 發行平台

- **Steam**
  - 上架費用：$100 USD
  - 分潤：70/30
  - 優點：流量、願望清單、演算法
  - 注意：審核、退款政策

- **itch.io**
  - 上架費用：$0
  - 分潤：開發者自訂（可達 100%）
  - 優點：獨立友善、PWYW
  - 注意：流量需自帶

- **其他平台**
  - GOG（邀請制）
  - Epic（邀請制）
  - Humble Store
  - 自架官網

### DLC 與追加內容

- **類型**
  - 追加劇情 / 路線
  - 美術包（桌布、設定集）
  - 原聲帶
  - 18+ 內容（作為單獨 DLC）

  - 自訂折扣

- **組合包**
