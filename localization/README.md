## 考量因素

### 語言密度與工作量

**文字量分級：**

| 類型 | 字數（日文） | 英文字數（約） | 說明 |
|------|-------------|---------------|------|
| 短篇 | 5,000 - 20,000 | 3,000 - 15,000 | 數小時體驗 |
| 中篇 | 20,000 - 80,000 | 15,000 - 60,000 | 主流獨立 VN |
| 長篇 | 80,000 - 200,000 | 60,000 - 150,000 | 商業 VN |
| 超長篇 | 200,000+ | 150,000+ | Key 社、TM 作品等 |

**翻譯成本粗估（2024 時點）：**

| 語言對 | 每詞/字價格（USD）| 說明 |
|--------|------------------|------|
| 日→英 | $0.08 - $0.15 | 專業譯者價格範圍 |
| 日→中 | $0.04 - $0.10 | 簡繁體類似 |
| 英→日 | $0.10 - $0.20 | 反向較貴 |
| 日→韓 | $0.06 - $0.12 | 韓語市場需求增加 |

*注意：VN 翻譯通常比一般翻譯貴，因為需要處理口癖、敬語、文化梗等特殊問題。*

**翻譯時間粗估：**

| 文字量 | 單人全職（約） | 說明 |
|--------|---------------|------|
| 20,000 字 | 2-4 週 | 含校對 |
| 50,000 字 | 1-2 月 | 含校對 |
| 100,000 字 | 3-5 月 | 含校對 |
| 200,000+ 字 | 6 月+ | 可能需要多人 |

### 目標語言市場

**Steam 語言統計（參考 2024 硬體調查）：**

| 語言 | 用戶比例 | VN 市場觀察 |
|------|----------|------------|
| 英語 | ~35% | 最大市場、必要性最高 |
| 簡體中文 | ~22% | 第二大、價格敏感 |
| 俄語 | ~11% | VN 接受度中等 |
| 西班牙語 | ~7% | VN 翻譯少、潛力市場 |
| 巴西葡萄牙語 | ~3% | 類似西班牙語 |
| 繁體中文 | ~1.3% | 台港澳、與簡體有重疊 |
| 日語 | ~2% | VN 原生市場、品質期望高 |
| 韓語 | ~3% | 近年 VN 興趣增加 |

**市場優先級考量：**
1. 英語 — 幾乎必須
2. 簡體中文 — 高 ROI 潛力
3. 日語 — 如果是西方 VN 進入日本市場
4. 其他 — 依作品類型與社群需求

### 翻譯方式選項

**自行翻譯：**
- 成本：$0（時間成本除外）
- 適合：雙語開發者、小型專案
- 風險：非母語翻譯品質可能有問題

**外包專業翻譯：**
- 成本：高（見上方估算）
- 適合：商業專案、有預算的獨立作品
- 優點：品質較穩定
- 找人管道：
  - [ProZ](https://www.proz.com/) — 譯者媒合平台
  - [Fiverr](https://www.fiverr.com/) — 自由接案平台
  - 專門遊戲翻譯社（如 Frognation、XLOC）

**社群翻譯：**
- 成本：低或免費
- 適合：有熱情粉絲群的作品
- 風險：品質不一、進度難控、可能中途放棄
- 成功案例：Katawa Shoujo 的多語言版本

**機器翻譯 + 人工校對：**
- 成本：中（校對費用）
- 適合：大量文本、低預算
- 風險：VN 特有問題（敬語、口癖）MT 處理差
- 玩家社群對「機翻」的負面觀感

### 技術實作

**引擎原生支援：**

| 引擎 | 多語言支援 | 說明 |
|------|-----------|------|
| Ren'Py | 內建翻譯系統 | `.rpy` → `.pot` → `.po` |
| Unity + Naninovel | 內建本地化 | Scripts + Managed Text |
| Godot + Dialogic | 插件支援 | Godot 本地化系統 |

**字型考量：**
- CJK（中日韓）字型檔案大
- 免費 CJK 字型：Noto Sans CJK、Source Han Sans
- 字型授權需確認

**文字檔案格式：**
- Ren'Py：`.po` 格式（業界標準）
- 純文字分離：方便非技術譯者
- Excel / Google Sheets：簡單但有版本控制問題

**動態文字問題：**
- 玩家名字變數：{player_name}
- 複數形式：英文 "1 item" vs "2 items"
- 性別詞：he/she/they

### 文化適應議題

**「翻譯」vs「在地化」：**
- 翻譯：保留原意，可能有文化隔閡
- 在地化：調整為目標文化，可能失去原味

**常見爭議點：**
- 保留日文稱呼（-san、-chan）還是翻譯
- 日式笑話 / 雙關語的處理
- 食物、節日等文化元素
- 敏感題材的地區差異

**審查與分級：**
- 中國：成人內容不可、政治敏感
- 德國：暴力內容審查
- 澳洲：特定類型內容禁止
- Steam 各地區的可見性差異
